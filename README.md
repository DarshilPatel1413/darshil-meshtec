# Mini User Dashboard

This project is a small React dashboard that displays users fetched from an API.

## Features

- Fetch users from API
- Loading and error handling
- Search users by name
- Sort users by name
- View user details in modal
- Add new user locally
- Form validation
- Local storage persistence
- Responsive layout

## Tech Used

- React (Functional Components)
- Hooks (useState, useEffect, useMemo)
- CSS

## Setup Instructions

1. Clone the repository

2. Install dependencies

npm install

3. Run project

npm start

## Assumptions

- New users are added only locally and not sent to API
- Company name for new users is set as "Local Company"
